# LeaveSystem
for zhangzhen
1、gradle的jar包依赖：包括spring、springmvc、mybatis和javaweb基础的一些包。
2、gradle的默认目录：
  src/main/java  java源代码
  src/main/resources  各种配置文件
  src/main/webapp  工程根目录
3、web.xml：包括spring、springmvc、编码方式。
4、applicationContext.xml  spring的主配置文件，这里主要配置mybatis的链接和事务管理.
5、springmvc-config.xml  springmvc的主配置文件
